Perguntas rápidas

Notas práticas para instalar e operar sem surpresa.

Precisa instalar Python?

Não para usar o app. O download é um `.exe` standalone gerado com PyInstaller.

O app guarda senha?

Ele lê configs locais das CLIs. Para Hostinger, salva o API token no arquivo oficial `.hostinger.yaml`.

Posso instalar uma CLI que não está na lista?

Sim. Use o campo de comando customizado para rodar um instalador confiável, como `winget`, `npm`, `pipx`, `scoop` ou `choco`.

Por que isso ajuda quem trabalha com IA?

Agentes usam GitHub, cloud, deploy, modelos, automações e tokens locais. O dashboard reduz o tempo perdido validando ambiente.

Dá para usar mais de uma conta?

Sim. gcloud e Firebase aparecem por conta. Outras CLIs dependem do suporte nativo de perfil/login.

O download é atualizado?

Esta landing aponta para o build atual publicado junto com a página. Novos builds substituem o arquivo.
